Text Shaper

Client-side JavaScript application to format a block of text to an image. Drag and drop the file below, or experiment with the built-in examples. Image must be a .png with transparency where you don't want text to appear.

The height and width is how many pixels make up a character in the output. If you set both to 1, each character in the text will map directly to each pixel in the input image. Monospace characters are roughly twice as tall as they are wide, so that's a good starting point.
